# Japanese Numbers



## Ordinal Numbers
Japanese ordinal numbers such as `Ipponme` and `Nihonme` are often prepended names or used to directly refer to Kata or Kihon within a set.

::::{tab-set}

:::{tab-item} Shuffled
```{code-cell} python
:tags: ["remove-input"]
from jupytercards import display_flashcards
import csv

file = "ordinal_numbers.csv"
with open(file, newline="", encoding="utf-8") as f:
    data = list(csv.DictReader(f, delimiter=';'))
display_flashcards(data, shuffle_cards=True)
```
:::
