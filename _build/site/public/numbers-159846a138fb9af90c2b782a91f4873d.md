# Japanese Numbers

## Numbers - Reading

Hiragana is included as a phonetic aid, when writing the numbers down kanji is used.
Note that `u` and `i` at the end of words are generally silent so six is spoken as rok but sixth-dan as rokudan.

::::{tab-set}

:::{tab-item} Shuffled
```{code-cell} python
:tags: ["remove-input"]
from jupytercards import display_flashcards
import csv

file = "numbers_reading.csv"
with open(file, newline="", encoding="utf-8") as f:
    data = list(csv.DictReader(f, delimiter=';'))
result = data.copy()
for d in data:
    result.append({'front': d['back'], 'back': d['front']})
data = result
display_flashcards(data, shuffle_cards=True)
```
:::

## Ordinal Numbers
Japanese ordinal numbers such as `Ipponme` and `Nihonme` are often prepended to names or used to directly refer to Kata or Kihon within a set.

::::{tab-set}

:::{tab-item} Shuffled

:tags: ["remove-input"]

    data = list(csv.DictReader(f, delimiter=';'))
