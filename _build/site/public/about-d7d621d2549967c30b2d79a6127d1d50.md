# A collection of tools to learn the theory

We are expected to pick up quite a bit of information during practice, such as commands, kata names and terms. Being able to quickly recall these helps free up mental space in the dojo for actual practice. The goal of this website is to help you study at home with classic tools such as flashcards and more experimental tools such as the shinpan game.
